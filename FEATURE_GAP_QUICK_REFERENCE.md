## 📋 READY-TO-USE STORIES

### Story 1: Split Payments
```
AS A passenger traveling with friends
I WANT to split the ride cost between multiple people
SO THAT we can pay fairly without cash exchanges

ACCEPTANCE CRITERIA:
- Can select "split payment" before booking
- Can invite 2-7 people via phone/email
- Each person pays equal share via their payment method
- Ride doesn't start until all payments confirmed
- Fallback: primary booker pays if others don't respond in 5min
```

### Story 2: GDPR Data Export
```
AS A user in the EU
I WANT to export all my personal data
SO THAT I can exercise my Right to Access

ACCEPTANCE CRITERIA:
- "Download My Data" button in settings
- Generates ZIP with JSON files (profile, rides, payments)
- Includes PDF summary
- Email link when ready (async process)
- Data includes: profile, ride history, payment history, support tickets
- Completes within 30 days (regulatory requirement)
```

### Story 3: In-App Chat
```
AS A passenger
I WANT to message my driver without sharing my phone number
SO THAT I can coordinate pickup without privacy concerns

ACCEPTANCE CRITERIA:
- Chat opens when ride is accepted
- Real-time message delivery
- Shows typing indicator
- Read receipts
- Can send text and emojis
- Can share current location pin
- Auto-translates if languages don't match
- Chat history saved for 90 days
```
