## 💰 Fare Calculation

**7-Step Process:**
1. Haversine distance
2. Duration estimation
3. Base components (base + distance + time)
4. Vehicle multiplier premium
5. Surge pricing
6. Goods/weight surcharge
7. 18% GST taxes

**Example:** Taxi 34.5km = ₹795.32
