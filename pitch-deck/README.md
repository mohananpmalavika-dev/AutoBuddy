# AutoBuddy Pitch Deck

A browser-friendly pitch deck generated from `ARCHITECTURAL_REPORT.md`.

## How to view

1. Open `deck.html` in any modern browser:
   - `pitch-deck/deck.html`
2. Use arrow keys (or controls) to navigate slides.

## Notes
- Uses Reveal.js from a CDN (no build step required).
- If your network blocks CDNs, download Reveal.js locally and update the CDN URLs in `deck.html`.
