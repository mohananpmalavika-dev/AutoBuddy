# 👥 PASSENGER MANUAL

## Getting Started as a Passenger

### Account Creation

1. **Download AutoBuddy App**
   - iOS: Available on Apple App Store
   - Android: Available on Google Play Store
   - Web: Visit autobuddy.com

2. **Sign Up Process**
   ```
   Tap "Sign Up" → 
   Enter phone number → 
   Verify OTP → 
   Enter name & email → 
   Set password → 
   Account created!
   ```

3. **Profile Setup**
   - Add profile picture
   - Add home/work addresses
   - Add emergency contact
   - Set payment method

### Your First Ride

#### Step 1: Open the App
- Launch AutoBuddy
- Allow location permission when prompted

#### Step 2: Enter Destination
- Tap search box at top
- Type or speak your destination
- Select from suggestions
- Confirm pickup location

#### Step 3: Choose Ride Type
```
Ride Options Available:
- BIKE: Affordable, 1 passenger
- ECONOMY: Budget-friendly, 4 passengers
- PREMIUM: Comfortable, 4 passengers
- XL: Spacious, 5+ passengers
```

#### Step 4: View Price & Book
- See estimated fare
- See estimated time
- Choose payment method
- Tap "Book Ride"

#### Step 5: Wait for Driver
- See driver details & rating
- Live tracking of driver location
- Driver will arrive within estimated time
- Chat with driver if needed

#### Step 6: Enjoy Your Ride
- Driver will pick you up
- Live GPS tracking throughout ride
- Emergency button available if needed
- Provide destination tips if needed

#### Step 7: Complete Ride
- Ride automatically ends at destination
- Rate driver (1-5 stars)
- Provide feedback
- View receipt

### Common Admin Tasks

#### Task 1: Approve Driver Application

```
Steps:
1. Go to Drivers → Pending Applications
2. Click on driver name
3. Verify documents:
   - License (valid date)

   - Insurance (active)

4. Review ratings/feedback
5. Click "Approve" or "Reject"
6. Add notes if needed
7. System sends notification to driver
```

#### Task 2: Handle Customer Complaint

```
Steps:
1. Go to Support → Open Tickets
2. Click on complaint
3. Review incident details
4. Check trip details
5. Review passenger/driver ratings
6. Process refund if needed:
   - Determine amount
   - Approve refund
   - Select refund method
   - Add refund reason
```

```
   - Minimum ride value
   - Valid ride types
   - Valid locations
6. Review & Publish
```

#### Task 4: Monitor Live Trips

```
Steps:
```

# ❓ FAQ

## General Questions

**Q: How do I create an account?**
A: Download app → Sign up with phone → Verify OTP → Complete profile

**Q: What payment methods are accepted?**
A: Wallet, Credit/Debit card, UPI, Cash (select cities)

**Q: Can I schedule rides in advance?**
A: Yes, tap calendar icon and select future date & time

**Q: How is the fare calculated?**
A: Base fare + per-km charge + per-minute charge (+ surge if applicable)

## Passenger FAQ

**Q: What if my driver is late?**
A: Contact driver via chat, check tracking. >5 min late = free cancellation

**Q: Can I change my destination?**
A: Yes, update destination in-app. Driver notified & fare recalculated

**Q: How do I get refund?**
A: Tap receipt → "Request refund" → Select reason → Submit. Refund in 24 hours

**Q: Is my ride safe?**
A: Yes, we have safety features: SOS button, ride tracking, driver verification

## Driver FAQ

**Q: When do I get paid?**
A: Weekly payouts on Mondays via bank transfer

**Q: What documents do I need?**
A: License, vehicle registration, insurance, pollution certificate, Aadhar

**Q: How do I improve my rating?**
A: Drive safely, be professional, keep vehicle clean, communicate well

**Q: Can I refuse a ride?**
A: Yes, you can decline. Too many declines affects your rating

## Admin FAQ

**Q: How do I approve drivers?**
A: Review documents, verify details, click "Approve"

**Q: Can I cancel trips?**
A: Yes, in Admin Dashboard → Trips → Select trip → Cancel (with reason)

**Q: How do I create promotions?**
A: Admin Panel → Promotions → New Promo → Fill details → Publish
